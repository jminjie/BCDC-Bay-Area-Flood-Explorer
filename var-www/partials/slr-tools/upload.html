User Data Upload

Import a KML file or an ESRI Shapefile

  • KML file can be in .kml or .kmz format.
  • Shapefiles must be zipped and include all .cpg, .dbf, .prj, .sbn, .shp, and .sbx files.
  • Attributes associated with the data will be ignored.
  • Maximum upload file size is 5MB.

Select your file and click "Upload File" to begin importing.
